Compraventa de viviendas en Espaa

Vamos a realizar un anlisis del mercado de la vivienda segn su regimen y estado de todo el pais con los datos que nos proporciona el gobierno.

En primer lugar vamos a leer los datos utilizados para este anlisis. Los datos utilizados son de: https://datos.gob.es/es/catalogo/ea0010587-compraventa-de-viviendas-segun-regimen-y-estado-etdp-identificador-api-49280

Vamos a definir primero los Regimenes y estados que nos podemos encontrar:

-Viviendas protegidas:Esta caracterstica implica que la vivienda est sujeta a un rgimen legal especial que busca garantizar el acceso a la vivienda digna para personas con rentas medias o bajas.

-Viviendas libres:La vivienda no tiene ninguna proteccin de preciopara el nuevo propietario.

-Vivienda usada:La vivienda ha sido habitada.

-Vivienda nueva:Vivienda de nueva construccin, no ha sido habitada.

# Necesitamos una funcin que nos rellene los decimales hasta 3

def format_value(value):
    if '.' in str(value):
        formatted_value = f"{value:.3f}"
        parts = formatted_value.split(".")
        integer_part = parts[0] 
        decimal_part = parts[1] 
        return int(integer_part + decimal_part)
    else:
        return int(value)

def format_value2(value):
    # Comprobar si el valor es una cadena
    if isinstance(value, str):
        # Eliminar el punto decimal si est presente y luego convertir a entero
       
        integer_part, decimal_part = value.split('.')
        decimal_part = decimal_part.ljust(3, '0')
        cleaned_value = integer_part + decimal_part
        return int(cleaned_value)
    else:
        # Si el valor no es una cadena, simplemente convertirlo a entero
        return int(value)
